# # Gym Email Agent

A Python-based email automation tool that classifies gym-related inquiries and automatically generates responses.

## Features

* Detects membership inquiries
* Detects personal training inquiries
* Generates automatic replies
* Handles general inquiries
* Uses keyword-based classification

## Technologies Used

* Python
* Functions
* Lists
* Loops
* Conditional Logic

## Example

Input:

I want to join your gym.

Output:

Category: membership

Auto Reply:
Thank you for contacting Minimum Strength Gym...
